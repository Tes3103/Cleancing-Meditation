## User Experiance (UX)
### Useer Stories
**Client Goals** 
* To be able to see the website on various devices. 
* To have an easy access for contact if they have any question.
* To make it more accessible for them to be 
part of the meditation community.
* To have more information on the technical aspect of the meditation process and the guidance they require.

**First Time Visitor Goals**
* As a visiting user, I want to have detail information on what really the meditation community is about.
* As a visiting user, I want to have a detil information regarding the purpose and different level of the cleancing meditation.
* As a visiting user, I want to have the social links to have an up-to-date information.

**Returning Visitors**
* I want to have a free trial session to test if it is going to work for me.
* I want to have a detail information how i can be part of the community.
* I want to know how the process will be from the start to the final level.
## Design
### Typography
The following fonts are used from Google Fonts:
* Roboto is used for the headings and menu,
* Loto is used for the paragraph,
* These two are used because they contrast and complement well when paired eachother and give an appealing presentation for the reader.

* Gray and darkgray are used for the heading, menu, paragraph and additional black and white.

All images are used from the free download section and are used to give a visual feedback and to witness the inclusiveness and also to show some features of the group guided meditation service also available for every one including kids school.

## Deployment & Local Development

**Deployment**

Github Pages was used to deploy the live website. The instructions to achieve this are below:

1. Log in (or sign up) to Github.
2. Find the repository for this project,Cleansing Meditation.
3. Click on the Settings link.
4. Click on the Pages link in the left hand side navigation bar.
5. In the Source section, choose main from the drop down select branch menu. Select Root from the drop down select folder menu.
6. Click Save. Your live Github Pages site is now deployed at the URL shown.

**Local Development**

**To Fork**

To fork Cleansing-meditation repository

1. Log in (sign-up) to Github.
2. Select the repository for this poject, cleansing-meditation
3. Click the fork button in the top right corner.

**To Clone**

1. Log in (or sign up) to GitHub.
2. Go to the repository for this project, Cleansing-meditation.
3. Click on the code button, select whether you would like to clone with HTTPS, SSH or GitHub CLI and copy the link shown.
4. Open the terminal in your code editor and change the current working directory to the location you want to use for the cloned directory.
5. Type 'git clone' into the terminal and then paste the link you copied earlier, then Press enter.
